请帮我开发一个"AI超级猎头工作台 V1"。

项目目标：\
这是一个只服务我个人使用的本地 Web 工作台，不做 SaaS，不做多人协作。目标是帮助我把多个来源的岗位统一录入岗位库，再把猎聘网候选人录入人才库，然后通过 AI 解析和匹配评分，快速判断"这个候选人还能推荐给哪些岗位"。

技术要求：

1.  使用 Next.js + TypeScript + SQLite。

2.  前端使用简单清爽的后台管理界面。

3.  后端可以先用本地 API Route 实现。

4.  数据库使用 Prisma + SQLite。

5.  第一版不做登录、不做权限、不做浏览器插件、不做自动打招呼。

6.  所有功能优先简单可用。

第一版需要实现的页面：

1.  首页看板

- 今日新增岗位数

- 今日新增候选人数

- 高匹配推荐机会

- 待跟进候选人

- 待推荐岗位

    字段：

- 企业名称

- 行业

- 来源：人力聚合平台 / 禾蛙网 / 手工录入

- 合作状态

- 备注

    字段：

- 企业名称

- 岗位名称

- 城市

- 薪资范围

- 学历要求

- 年龄要求

- 行业要求

- 技能关键词

- 硬性要求

- 加分项

- 排除项

- JD原文

    字段：
