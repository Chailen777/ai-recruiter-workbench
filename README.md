# AI超级猎头工作台 V0.1

这是一个只服务个人本地使用的 Web 工作台，用于录入企业、岗位、候选人，并通过规则引擎查看候选人与岗位的匹配分。

## 技术栈

- Next.js + TypeScript
- Prisma + SQLite
- 本地 App Router 页面和 Server Actions
- 第一版不包含登录、权限、浏览器插件和自动打招呼

## 本地运行

1. 安装依赖

```bash
npm install
```

如果在 Windows PowerShell 中遇到 `npm.ps1` 执行策略限制，可以改用：

```bash
npm.cmd install
```

2. 初始化数据库

```bash
npm.cmd run prisma:migrate -- --name init
```

如果 Prisma engine 下载较慢或失败，也可以使用本项目提供的 Node SQLite 初始化脚本：

```bash
npm.cmd run db:init
```

3. 启动开发服务

```bash
npm.cmd run dev
```

打开 http://localhost:3000。

## 功能范围

- 首页看板：今日新增岗位、今日新增候选人、高匹配机会、待跟进候选人、待推荐岗位
- 企业库：企业名称、行业、城市、来源、合作状态、备注的新增、查看、编辑、删除
- 岗位库：企业、岗位名称、城市、薪资、学历、年龄、经验、行业、技能关键词、硬性要求、加分项、排除项、JD 原文、状态、来源的新增、查看、编辑、删除
- 人才库：姓名、当前职位、当前公司、城市、学历、年龄、工作年限、期望薪资、技能标签、行业背景、简历原文、沟通状态、备注的新增、查看、编辑、删除
- 匹配评分：选择候选人后，对所有岗位计算匹配分并从高到低排序

## 匹配规则

总分 100 分：

- 学历符合：20 分
- 年龄符合：15 分
- 城市符合：10 分
- 工作年限符合：15 分
- 行业符合：20 分
- 技能关键词匹配：20 分

评分规则在 `src/lib/matching.ts` 中，后续可以替换为 AI 解析或更细的规则。
